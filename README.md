# javascript-api
重读 JavaScript 的一些主要 api

## 开始

```
// Project setup
git clone https://github.com/sjx1995/javascript-api.git

// Compiles and hot-reloads for development
npm run dev

// Compiles and minifies for production
npm run build

// Run your tests
npm run test
```
